# 🧼 工业级大模型监督微调 (SFT) 数据集自动蒸馏与格式化转换系统

本项目聚焦于大模型商用落地（LLM Alignment）中最核心、且工程容错率极低的 **微调数据集清洗与特征格式化重构（Data Engineering for SFT）** [1.3]。全盘淘汰了效率低下的纯人工打标，实现了全自动的“赛博知识蒸馏流水线” [1.3]。

## 🌟 核心技术链路 (Architecture)
1. **多源数据收流**：无缝承接前置 RAG 系统的文本清洗内核，读取本地学术 PDF 提取出脱水 25% 的黄金知识切片 [1.3]。
2. **大模型反向自生成 (LLM-as-a-Generator)**：调度智谱 `glm-4-flash` 大脑作为“出题专家”，依据论文切片毫秒级反向蒸馏出契合人类提问逻辑的高质量 Q&A 对 [1.3]。
3. **标准三元组格式化 (JSONL Layout)**：严格将数据重构为 OpenAI 兼容的 `{"messages": [{"role": "system"...}, {"role": "user"...}, {"role": "assistant"...}]}` 训练集格式 [1.3]。
4. **硬核数据审计 (Data Audit)**：利用流式零缩进写入控制（`json.dumps`），附加 2048 令牌长度阈值强行卡死过滤，100% 封杀数据读取器（Dataloader）因格式穿帮引发的 CUDA 显存崩溃隐患 [1.3]。
